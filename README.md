# IT Admin Toolkit - Batch File

A powerful, portable, and easy-to-use Windows IT Administrator Toolkit built using Batch (.bat) file.

Designed for quick access to all essential Windows administrative tools, diagnostics, and system utilities from a single menu.

---

## ✨ Features

- **Secure Login System** (Username + Password)
- Clean and organized menu interface
- One-click access to 24+ essential admin tools
- System Diagnostics
- Service & Management Tools
- Security & Network Tools
- Fully portable (No installation required)
- Runs with Administrator privileges

---

## 📋 Available Tools

**System & Diagnostics**
- System Information
- Disk Drive Status
- System File Checker (SFC)
- Check Disk (CHKDSK)
- Disk Management
- Reliability Monitor
- Event Viewer
- Task Manager

**Management & Configuration**
- Computer Management
- Services
- System Configuration (msconfig)
- Group Policy Editor
- Registry Editor
- Device Manager
- Programs and Features
- Local User Management

**Security & Network**
- Windows Security
- Windows Defender Firewall
- Windows Update
- Network Information (ipconfig)
- Network Connections
- Command Prompt
- PowerShell
- DirectX Diagnostic Tool

---

## 🚀 How to Use

1. Download `IT_Toolkit.bat`
2. **Right-click** on the file → **Run as administrator**
3. Login with default credentials:
   - **Username:** `admin`
   - **Password:** `123`
4. Select any tool from the menu

---

## 🔐 Default Credentials

- **Username:** `admin`
- **Password:** `123`

> ⚠️ **Important:** Change the username and password immediately after downloading for security.

---

## 🔧 How to Change Password

Open the `.bat` file in Notepad and edit these lines at the top:
